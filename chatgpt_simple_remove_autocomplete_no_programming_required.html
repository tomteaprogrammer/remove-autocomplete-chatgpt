ChatGPT Simple Bookmarklets

How to install

Drag a button below to your bookmarks bar, or right-click a button and choose “Bookmark this link.” After it is saved, click the bookmark on the target site to remove the suggestion list and keep it removed.

ChatGPT Simple ChatGPT Modified

Which button to use

ChatGPT Simple: Use this first. It targets the exact suggestion list you described and avoids removing other lists on the page.
ChatGPT Modified: Use this if the site changes class names and the first button stops working.

Technical details

ChatGPT Simple uses the selector: ul.w-full.flex-col.p-2\.5.max-sm\:px-0.flex (dots and colons are escaped). It removes all matches, then attaches a MutationObserver to remove any reinserted matches.

ChatGPT Modified uses the looser selector: ul.w-full.flex-col. This is resilient to class changes but may match other ULs using those utility classes.

Both bookmarklets run only when you click the bookmark. They do not install anything and they do not persist across page reloads automatically.
